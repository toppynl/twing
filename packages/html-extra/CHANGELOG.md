# @toppynl/twing-html-extra

## 0.0.0

### Minor Changes

- Initial release: port of `twig/html-extra`.

  - `html_classes` function: conditional CSS class builder
  - `html_attr` function: render HTML attribute string from object
  - `html_attr_merge` filter: merge two attribute objects
  - `html_attr_type` helper
  - `html_cva` function: class-variance-authority style variant builder
